# rivi_assignment

## Getting Started

1. clone repository

```
git clone https://github.com/deveshagnihotri/rivi_assignment.git
```

2. Install package

```
npm install
```

3. Start Application

```
npm start
```

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
